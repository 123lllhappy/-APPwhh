## 「我很好」APP 功能介绍

### 📱 应用目的
这是一款**个人安全守护应用**，帮助用户在紧急情况下快速求助，并通过定时打卡机制保障用户安全。

---

### ✨ 核心功能

**1. 每日打卡**
- 点击「今日打卡」按钮完成每日打卡
- 打卡卡片支持正反面切换（点击切换）
- 可自定义正面/背面图片（支持随机或顺序展示）

**2. 紧急求助 (SOS)**
- 点击 SOS 按钮立即向所有紧急联系人发送求助短信
- 短信包含当前位置信息和地图链接
- 支持自定义短信模板

**3. 自动求助**
- 设置超时时间（1/12/24/48/72小时）
- 超过设定时间未打卡自动发送求助短信
- 后台守护服务持续运行

**4. 骑行守护**
- 专门的骑行模式
- 骑行过程中自动检测异常

**5. 联系人管理**
- 添加/编辑/删除紧急联系人
- 支持设置关系（家人/朋友/其他）

**6. 历史记录**
- 查看所有打卡记录
- 查看求助历史（SOS/自动求助）

**7. 设置项**
- 开机自启动
- 超时时间设置
- 短信模板管理
- 图片设置（正反面图片选择）

---

### 🚀 使用方法(在文件下目录最下面即可下载安卓APK应用)

**首次使用：**
1. 启动应用，授予必要权限（定位、短信、通知）
2. 添加紧急联系人
3. 设置自动求助时间

**日常使用：**
- 每天打开应用点击「今日打卡」
- 遇到紧急情况点击底部 SOS 按钮
- 需要骑行时开启骑行模式

---

### ⚠️ 重要提示
- 需保持应用在后台运行（后台程序关闭将导致自动求助和骑行求助无法使用）
- 首次使用需添加至少一个紧急联系人
- 建议开启「开机自启动」确保守护服务持续运行
